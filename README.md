# Portfolio MVP - The Kinetic Architect
Designed to be an interactive Next.js application built with React Three Fiber, Supabase backend, and Tailwind.

## Setup
1. Populate your `.env.local` file with Supabase and GitHub credentials.
2. Run `npm install` (Use `--legacy-peer-deps` if React 18 conflicts with R3F versions).
3. Ensure Supabase schema represents `projects`, `contact_messages`, and `resumes` properly.
4. Run `npm run dev`

Navigate your browser to `http://localhost:3000` to interact with your dashboard. Administrative portal resides at `/admin/login`.
